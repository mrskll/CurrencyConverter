# 💱 Конвертер валют

## Описание проекта
Приложение для конвертации валют с получением актуальных курсов из внешнего API.

## Стек технологий
- **Язык:** C# (.NET Framework)
- **Интерфейс:** Windows Forms (WinForms)
- **API:** ExchangeRate-API (бесплатный, без ключа)
- **Система контроля версий:** Git, GitHub

## Функционал
- Получение реальных курсов валют из интернета
- Конвертация RUB, USD, EUR, GBP, CNY
- Графический интерфейс WinForms
